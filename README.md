## 💡 介绍
1. 两种方式登录QQ空间提取SKEY
2. 这个是模拟QQ空间登录获取登录信息的程序，有密码登录和扫码登录2种方式，可同时获取skey、P_skey、superkey。
## 🍄 安装
直接上传压缩包并解压 访问域名即可
